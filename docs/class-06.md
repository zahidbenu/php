# পিএইচপি Array ফাংশন

## in_array() ফাংশন

এই ফাংশন দিয়ে আমরা কোন একটি এলিমেন্ট একটি অ্যারের মধ্যে আছে কিনা তা চেক করতে পারি। এই ফাংশনের প্রথম প্যারামিটার হচ্ছে এলিমেন্ট এবং দ্বিতীয় প্যারামিটার হচ্ছে অ্যারে। এই ফাংশন যদি এলিমেন্ট অ্যারের মধ্যে পায় তাহলে এটি ট্রু রিটার্ন করবে আর না পাইলে ফলাফল হবে ফলস।

```php
<?php
$fruits = array('apple', 'banana', 'orange');

if (in_array('apple', $fruits)) {
    echo 'Apple exists';
} else {
    echo 'Apple does not exist';
}
```

আউটপুট:

```php
Apple exists
```

আরেকটা উদাহরণ

```php
<?php

$allowedFileExtensions = ['jpg', 'jpeg', 'png', 'gif'];

$fileName = 'image.png';

$fileParts = explode('.', $fileName);

$fileExtension = strtolower(end($fileParts));

if (in_array($fileExtension, $allowedFileExtensions)) {
    echo 'File is allowed';
} else {
    echo 'File is not allowed';
}
```

আউটপুট:

```php
File is allowed
```

## array_keys ফাংশন

এই ফাংশনের মাধ্যমে আমরা কোন একটি অ্যারের সব কী গুলো পেতে পারি। এই ফাংশনের প্রথম প্যারামিটার হচ্ছে অ্যারে এবং দ্বিতীয় প্যারামিটার হচ্ছে অপশনাল। যদি আমরা দ্বিতীয় প্যারামিটার দেই তাহলে এই ফাংশন সেই অ্যারের মধ্যে সেই ভ্যালু গুলোর কী গুলো দেখাবে।

```php
<?php
$fruits = array('apple', 'banana', 'orange');

$keys = array_keys($fruits);

print_r($keys);
```

আউটপুট:


## array_map ফাংশন

এই ফাংশনের মাধ্যমে আমরা কোন একটি অ্যারের সব এলিমেন্ট গুলোকে একটি ফাংশনের মধ্যে পাঠাতে পারি। এই ফাংশনের প্রথম প্যারামিটার হচ্ছে একটি ফাংশন এবং দ্বিতীয় প্যারামিটার হচ্ছে অ্যারে। এই ফাংশন সব এলিমেন্ট গুলোকে সেই ফাংশনের মধ্যে পাঠায়।

```php
<?php
```

## array_filter ফাংশন

এই ফাংশনের মাধ্যমে আমরা কোন একটি অ্যারের সব এলিমেন্ট গুলোকে একটি ফাংশনের মধ্যে পাঠাতে পারি এবং সেই ফাংশনের মধ্যে যেই কন্ডিশন থাকবে সেই কন্ডিশন অনুযায়ী ফিল্টার করতে পারি। এই ফাংশনের প্রথম প্যারামিটার হচ্ছে অ্যারে এবং দ্বিতীয় প্যারামিটার হচ্ছে একটি ফাংশন। এই ফাংশন সব এলিমেন্ট গুলোকে সেই ফাংশনের মধ্যে পাঠায় এবং সেই ফাংশনের মধ্যে যেই কন্ডিশন থাকবে সেই কন্ডিশন অনুযায়ী ফিল্টার করে ফেলে।

```php
<?php
$numbers = array(1, 2, 3, 4, 5);

$evenNumbers = array_filter($numbers, function ($number) {
    return $number % 2 == 0;
});

print_r($evenNumbers);
```

## array_walk ফাংশন

এই ফাংশনের মাধ্যমে আমরা কোন একটি অ্যারের সব এলিমেন্ট গুলোকে একটি ফাংশনের মধ্যে পাঠাতে পারি। এই ফাংশনের প্রথম প্যারামিটার হচ্ছে অ্যারে এবং দ্বিতীয় প্যারামিটার হচ্ছে একটি ফাংশন। এই ফাংশন সব এলিমেন্ট গুলোকে সেই ফাংশনের মধ্যে পাঠায়।

```php
<?php

array_walk($fruits, function (&$fruit) {
    $fruit = strtoupper($fruit);

print_r($fruits);
```

## usort ফাংশন

এই ফাংশনের মাধ্যমে আমরা কোন একটি অ্যাসোসিয়েটিভ অ্যারের সব এলিমেন্ট গুলোকে সর্ট করতে পারি। এই ফাংশনের প্রথম প্যারামিটার হচ্ছে অ্যাসোসিয়েটিভ অ্যারে এবং দ্বিতীয় প্যারামিটার হচ্ছে একটি ফাংশন। এই ফাংশন সব এলিমেন্ট গুলোকে সেই ফাংশনের মধ্যে পাঠায় এবং সেই ফাংশনের মধ্যে যেই কন্ডিশন থাকবে সেই কন্ডিশন অনুযায়ী সর্ট করে ফেলে।
